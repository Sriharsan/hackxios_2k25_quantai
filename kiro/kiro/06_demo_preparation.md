# Demo Prep - Getting Ready for Judges

## The 5-Minute Story

**Opening (30 seconds):**
"Portfolio managers handle $2 trillion in assets, but clients can't verify that historical performance data is real. This trust gap costs billions in compliance and creates massive legal risks. We built the first system that provides cryptographic proof of every investment decision."

**Live Demo (3 minutes):**
1. Create a pension fund portfolio using AI optimization
2. Show the institutional-grade analytics and risk metrics  
3. Click "Record on Blockchain" - live transaction to Ethereum
4. Open Etherscan to show the immutable proof
5. Explain how anyone can verify this independently

**Business Impact (1 minute):**
"This solves real problems for pension funds, asset managers, and regulators. We're not just building software - we're rebuilding trust in financial services through math instead of promises."

**Wrap-up (30 seconds):**
"Questions?"

## Technical Setup

**Demo Environment:**
- Streamlit app running locally (more reliable than cloud)
- Sepolia testnet account funded with 0.01 ETH
- Browser tabs pre-opened (app + Etherscan)
- Backup internet connection ready

**Demo Data:**
- Sample pension fund portfolio (60% stocks, 30% bonds, 10% alternatives)
- Pre-calculated optimization results for faster demo
- Known transaction hashes for backup scenarios

## Backup Plans

**If Blockchain Fails:**
Show pre-recorded transaction and explain the process. Have Etherscan link ready to show existing anchored portfolios.

**If App Crashes:**
Switch to backup demo video that shows complete workflow.

**If Network Issues:**
Use mobile hotspot and simplified offline presentation.
